# Assets

Coloca aqui los recursos de produccion.

- `models/`: modelos `.glb` o `.gltf`.
- `targets/`: archivo `.mind` del Image Target.

Los nombres usados por defecto en el prototipo son:

- `assets/targets/realalto-target.mind`
- `assets/models/welcome.glb`
- `assets/models/valdivia.glb`
- `assets/models/excavacion.glb`
- `assets/models/ceramica.glb`
- `assets/models/marker.glb`
